HOW TO DEODEX ANY ROM

For stock/ custom ROMs running Android 6.0 to 8.1.0 without PC (few custom rom 9.0 are supported)

Requirements:

  • Phone rooted by Magisk.
  • TWRP (optional but preferred for back-up and safety).
  • P3C apk with extra odex.zip.
  • Patience.

Reminder:

Take a back-up of you ROM before trying to Deodex. If anything goes wrong you can revert back easily.

Steps:

  1. Download the ZIP file provided above and extract it.
  2. You'll find 2 zip files inside it with name extra_odex.zip and extra_odex_v4.zip. Copy them to your "Internal storage/ Download" folder. (Don't extract these 2). So the both zip files will be in download folder.
  3. Install the 3C All-in-One Toolbox Pro apk from the extracted Zip file.
  4. Open the App and grant magisk su permission when asked.. or open magisk and grant it from #Superuser.
  5. Grant storage permission and other permission the app asks for.
  6. Swipe from left and tap On "Device Manager".
  7. Now tap on "One-Click" tab on top section of screen in the app.
  8. If you have granted root properly you can see "(de)Odex ROM" option tap on it.
  9. Now you can see few options set them as below:
    • "/system/app : OFF"
    • "/system/priv-app : OFF"
    • "/system/framework : ON"
    • "Rebuild classes.dex if needed : ON"
    • "Zip-align system APKs : ON"
      • Basically last 3 button should be ON with operation set to Deodex).
  10. Now click on OK and keep it on charging. (Will take 45mins to 3 hours depends on ROM so make sure the app never get closed.)
  11. When you get notification or in app pop-up Process Completed, it'll ask to reboot. Click on reboot/ ok.
  12. Reboot can take up to 5-7 mins don't panic. After the device is rebooted your ROM is now Deodexed.

NOTE:

  • If process fails after 70%-80%, reboot and try from step 6.
  • If the process fails before 70% then clear data of 3C app and follow from step 3.
  • If process gets failed instantly, then make sure the extra_odex.zip is placed correctly, if it is then you need to change your ROM.
  • If this method didn't work for you then let us known your device details at support channel.
