ES FILE EXPLORER: ROOT-MODE

Steps:

  1. Open ES File Explorer app.

  2. Press the Fast Access Menu button (top left - 3 lines).

  3. Press Root Explorer and this will switch on Root Explorer.

  4. Root browsing is now enabled.

  5. Super User may now open and ask if you wish to grant ES access to Root.

  6. Select Grant or Allow.

NOTE:

You can also grant root access thru magisk under Superuser.
