MAGISK MODULE- APP SYSTEMIZER

Systemizes user-installed apps systemlessly thru magisks! Supports all device running Magisk.

How to Install:

  1. Open Magisk Manager> Menu icon> Downloads (search for App Systemizer (terminal emulator), tap Download button> tap install then reboot phone.

  2. Download and install Terminal Emulator for Android app by Jack Palevich from Playstore.

  3. Run Terminal Emulator then enter "su". (grant superuser access when prompted!)

  4. Enter "systemize -f", follow on-screen instructions.

  5. Reboot ur phone.

NOTE:

select: /system/priv-app

UPDATE:

In order for App Systemizer v17.2.2 to work, you also need to install Busybox for android SDk module by osm0sis.
