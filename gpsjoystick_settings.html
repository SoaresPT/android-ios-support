GPS JOYSTICK SETTINGS

Various settings for GPS Joystick fake gps app.
PHONE:
  1. developer options> mock location app (urfakegpsapp)
  2. set gps to high accuracy
  3. reboot phone
GPS Location - GPS Joystick:
  1. enable suspended mocking=OFF
  2. enable auto suspend location=OFF
  3. enable indirect mocking=ON
  4. enable system mode=OFF
  5. disable location=OFF
  6. enable a-gps reset=OFF
